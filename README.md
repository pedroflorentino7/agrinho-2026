👨‍🌾 Créditos e Inspiração
Desenvolvido para o projeto Toma Agro Forte - uma iniciativa que promove o diálogo entre produção agrícola e sustentabilidade ambiental.

A mecânica do jogo é inspirada em desafios reais do agronegócio brasileiro:

O dilema entre aumentar produtividade e preservar o meio ambiente

Soluções tecnológicas que beneficiam ambos os lados

Práticas sustentáveis como rotação de culturas e reflorestamento
